# AdvancedTodo
A C# GUI project made with the WinForms framework (IN DEVELOPMENT)

## AdvancedTodo V1.1
- Added widgets and a left sidepanel to the main form (`Add Task`, `Remove Task`, `Shop`, `Profile`, `Settings`)
- Added `add task` button functionality
- Integrated the ability to choose from dark to light theme at the first time of logging in

## AdvancedTodo V1.2
- Removed the button `Remove Task` from the left sidepanel
- Added `Shop` button functionality and a system
- Added `Profile` button functionality and a system
- Added `Settings` button functionality, only the UI is half-done, the system is being constructed
- Added a `badge` system (if you complete specific tasks, you earn badges that gets put onto your profile)
- Added the ability to buy **Themes**, **Username Colors** and **Titles** with TDC (To-Do Coins) and TTC (Total Tasks Completed).
